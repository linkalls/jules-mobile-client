# Dark Theme Screenshots Needed

This directory should contain dark theme screenshots for:

1. **sessions-dark.png** - Sessions list in dark mode
2. **new-task-dark.png** - New task creation screen in dark mode
3. **settings-dark.png** - Settings screen in dark mode
4. **session-detail-dark.png** - Session detail/chat screen in dark mode (bonus)

## How to Generate Screenshots

1. Run the app on a device or simulator
2. Enable dark mode in Settings
3. Navigate to each screen and take screenshots
4. Save with the naming convention: `[screen-name]-dark.png`
5. Update README.md to include dark theme screenshots

## Screenshot Specifications

- Format: PNG
- Dimensions: 472 x 1024 (to match existing screenshots)
- Color depth: 8-bit/color RGBA
- Theme: Dark mode enabled

Note: The app fully supports dark mode with auto theme detection and manual toggle.
